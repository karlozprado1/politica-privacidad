Política de Privacidad

Última actualización: 14 de mayo de 2025

Esta política de privacidad describe cómo la aplicación Pelixplay ("nosotros", "nuestra", "la app") recopila, utiliza y protege la información de los usuarios.

1. Información que recopilamos

La aplicación no recopila información personal identificable como nombre, correo electrónico o número de teléfono.

Podemos recopilar información no personal automáticamente, como el modelo del dispositivo, sistema operativo y estadísticas anónimas de uso para mejorar la experiencia del usuario.

2. Uso de la información

Utilizamos la información recopilada para:

  • Mejorar el funcionamiento de la app.
  • Corregir errores técnicos.
  • Entender cómo los usuarios interactúan con la aplicación.

3. Contenido de terceros

Esta app utiliza la API de The Movie Database (TMDb) para mostrar información pública sobre películas, series, trailers e imágenes.

Esta aplicación utiliza la API de TMDb pero no está avalada ni certificada por TMDb.

Podemos incluir enlaces a sitios de terceros como YouTube. No controlamos esos sitios ni su contenido.

4. Publicidad (si aplica)

La app puede mostrar anuncios a través de servicios de terceros como Google AdMob, que podrían usar identificadores anónimos para mostrar anuncios relevantes. No se comparte información personal con redes publicitarias.

5. Seguridad

Aplicamos medidas razonables para proteger los datos no personales recopilados. Sin embargo, ninguna transmisión por Internet es completamente segura.

6. Cambios a esta política

Nos reservamos el derecho de modificar esta política en cualquier momento. Te notificaremos publicando la versión actualizada en esta misma página.

7. Contacto

Si tienes preguntas sobre esta política, puedes escribirnos a: pelixplay.com@gmail.com
